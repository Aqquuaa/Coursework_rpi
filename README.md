# UAV Optical Targeting System
Програмна система розрахунку географічних координат наземної цілі в режимі реального часу для бортових мікрокомп'ютерів БПЛА. 

Проєкт розроблено з урахуванням обмежень (Raspberry Pi 4, 1GB RAM) і базується на концепції Edge Computing — уся обробка відеопотоку та телеметрії відбувається безпосередньо на борту літального апарата для усунення мережевих затримок.

## Основні характеристики

- **Асинхронна обробка даних:** Паралельне зчитування телеметрії MAVLink та захоплення відеопотоку у різних потоках.
- **Оптимізований Ray Marching:** Використання алгоритму трасування променів для знаходження точки перетину оптичної осі камери з рельєфом.
- **Пряма інтеграція DEM:** Збереження цифрової моделі рельєфу (SRTM) безпосередньо в оперативну пам'ять.
- **HTTP Трансляція:** Тестовий вебсервер на базі Flask для трансляції відеопотоку з накладеним HUD по бездротовій мережі.

## Апаратні вимоги

- Мікрокомп'ютер: Raspberry Pi 4 1GB RAM+ (або аналогічний пристрій).
- Польотний контролер: Будь-який сумісний з протоколом MAVLink (на базі ArduPilot).
- Камера: Будь-яка USB-камера.
