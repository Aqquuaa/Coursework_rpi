# Інструкція зі встановлення та запуску

## Системні вимоги
- Python 3.7+
- Операційна система на базі Linux (рекомендовано Raspberry Pi OS / Raspberry Pi OS Lite).
- Підключений польотний контролер на Baud 115200 (порт за замовчуванням: `/dev/ttyACM0`) та USB-камера.

## Залежностей
Відкрийте термінал у папці проєкту та виконайте команду для встановлення всіх необхідних бібліотек:

    pip install Flask opencv-python numpy pymavlink

## Підготовка цифрової моделі рельєфу (DEM)
Завантажте файл матриці висот SRTM для вашого регіону (наприклад, `N50E030.hgt`) з сайту NASA у кореневу директорію скрипта. Якщо файл відсутній, програма автоматично згенерує резервну плоску поверхню на висоті 150 м для можливості тестування алгоритмів.

## 4. Запуск системи
Виконайте команду запуску:

    python3 main.py

Після успішного запуску та підключення до контролера (повідомлення `[MAVLINK] Heartbeat detected!`), трансляція відео з накладеним інтерфейсом (HUD) буде доступна у браузері за адресою:

    http://<IP_адреса_Raspberry_Pi>:5000/
